Detailed Description

Основной интерфейс логгера, позволяющий записывать сообщения в различные источники (файл или сокет)

Класс реализует обёртку над сессией записи логов (Session) и предоставляет методы открытия/закрытия сессии, а также записи лог-сообщений с заданным уровнем
