template<typename T >
std::optional< T > Logger::Logger_protocol::extract_last_number ( std::string &  entry)

Извлекает последнее число из строки и удаляет его

разделяет последнее число и строку по пробелу

Template Parameters
TТип числа для извлечения (int, long, double)
Parameters
entryСтрока, из которой извлекается число. После вызова число будет удалено из строки
Returns
optional<T> Извлечённое число, либо пустое значение, если число не найдено

Алгоритм:

  • Удаляются завершающие пробелы
  • Ищется последний пробел в строке
  • После пробела ожидается число, парсится через std::from_chars
  • Из строки вырезается пробел и число
  • В случае ошибки возвращается пустой std::optional
